# Обновление переменных окружения в Vercel

## Текущая конфигурация

После изменений в коде необходимо обновить переменные окружения в Vercel:

### 1. TELEGRAM_ADMIN_CHAT_ID
**Значение:** `120352240`

**Назначение:**
- Проверка доступа к командам бота (`/start`, `/stop`, `/status` и т.д.)
- Проверка доступа к кнопкам "Одобрить"/"Отказать"

**Как обновить:**
1. Vercel Dashboard → Ваш проект → Settings → Environment Variables
2. Найдите `TELEGRAM_ADMIN_CHAT_ID`
3. Измените значение на `120352240`
4. Сохраните

### 2. TELEGRAM_PUBLISH_GROUP_ID
**Значение:** `4993347411`

**Назначение:**
- Отправка карточек с кнопками "Одобрить"/"Отказать" для одобрения событий
- Публикация одобренных событий в группу

**Как проверить/установить:**
1. Vercel Dashboard → Ваш проект → Settings → Environment Variables
2. Проверьте, что `TELEGRAM_PUBLISH_GROUP_ID` установлен
3. Если нет - добавьте со значением `4993347411`
4. Сохраните

## Роли аккаунтов

### 7007868967 (@yummspb)
- **Роль:** Мониторинг каналов через Client API
- **Где:** Worker на Render.com
- **Что делает:** 
  - Мониторит каналы конкурентов
  - Передает сообщения боту через webhook для анализа

### 120352240
- **Роль:** Администратор бота
- **Где:** Группа 4993347411
- **Что делает:**
  - Использует команды бота (`/start`, `/stop`, `/status`)
  - Нажимает кнопки "Одобрить"/"Отказать" на карточках событий

## После обновления

1. Дождитесь редеплоя Vercel (или сделайте Redeploy вручную)
2. Проверьте команды:
   - Отправьте `/start` с аккаунта 120352240 - должно работать
   - Отправьте `/start` с аккаунта 7007868967 - должно быть "Доступ запрещен"
3. Проверьте карточки:
   - Опубликуйте сообщение в мониторируемом канале
   - Карточка с кнопками должна появиться в группе 4993347411
   - Кнопки должен нажимать только 120352240
