# Исправление ошибки 401 в логах

## Проблема

Страница `/admin/logs` возвращает 401 Unauthorized - это означает, что вы не авторизованы в админ-панели.

## Решение

### Шаг 1: Авторизуйтесь в админ-панели

1. Откройте: `https://chatbot-tg.vercel.app/admin/login`
2. Введите логин и пароль администратора
3. После успешного входа вы будете перенаправлены на главную страницу админ-панели

### Шаг 2: Откройте страницу логов

После авторизации откройте: `https://chatbot-tg.vercel.app/admin/logs`

Теперь логи должны загружаться без ошибки 401.

## Альтернатива: Проверка логов через Vercel Dashboard

Если не хотите авторизовываться, можете смотреть логи напрямую в Vercel:

1. Откройте Vercel Dashboard → ваш проект
2. Перейдите в раздел **Logs**
3. Там будут все логи в реальном времени

## Что видно в логах Vercel

Из ваших логов видно:
- ✅ Webhook получает сообщения: `POST 200 /api/tg/webhook`
- ✅ Сообщения обрабатываются: `✅ WEBHOOK: processed in 915ms`
- ❌ Ошибка отправки карточки: `chat not found` с ID `4993347411`

**Проблема:** ID группы должен быть `-4993347411` (с минусом), а не `4993347411`.

## Исправление ID группы

После обновления кода (который автоматически добавляет минус), проверьте в Vercel:

1. Vercel Dashboard → Settings → Environment Variables
2. Найдите `TELEGRAM_PUBLISH_GROUP_ID`
3. Убедитесь, что значение: `-4993347411` (с минусом)
4. Если нет минуса - добавьте его
5. Передеплойте проект

После этого карточки должны отправляться успешно.
